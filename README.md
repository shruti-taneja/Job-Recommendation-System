# Job Recommendation System

## Overview

A console-based Job Recommendation System developed using Core Java, OOP, JDBC, MySQL, SQL, and DAO Pattern. The application allows users to register, log in, and receive skill-based job recommendations. It also provides role-based access for administrators to manage recommendation data.

## Features

- User Registration
- User Login Authentication
- Email Validation
- Skill-Based Job Recommendation
- CRUD Operations
- Role-Based Authentication (Admin/User)
- JDBC Database Connectivity
- DAO Design Pattern

## Technologies Used

- Core Java
- Object-Oriented Programming (OOP)
- JDBC
- MySQL
- SQL
- DAO Pattern
